# MSE_MA_VI_Projet
Projet de Visualisation

Analyse de données provenant des matchs d'e-sport professionnels sur le jeu Counter-Strike 2. Les données proviennent du site hltv.org, site référence dans l'éco-système Counter-Strike 2.

Les analyses statistiques et visuelles s'adressent aux joueurs et visionneurs de matchs professionnels réguliers ou occassionnels de tous niveaux.
